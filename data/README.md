# Data directory

Raw NYBB images and annotations are not included in this public-ready package.

Recommended local layout:

```text
data/
├── raw/
│   ├── images/
│   └── annotations/
├── processed/
│   ├── images/
│   ├── instance_maps/
│   ├── type_maps/
│   └── metadata.csv
└── splits/
```

Do not upload private images, annotations, checkpoints, or embeddings to a public repository unless you have permission.
